\doxysubsection{Detailed Description}
A class to perform CLI. 

The class provides commandline interface to interract with \doxylink{classMemoryAccessor}{Memory\+Accessor} effectively. It can handle various commands with arguments. The class can either execute a command as a std\+::string or read a command from stdin. Moving, copying and creating more than 1 instance of the class is prohibited.
