\doxysubsection{Detailed Description}
A struct that represents command in \doxylink{classConsole}{Console}. 

In this struct all the variables related to \doxylink{classConsole}{Console} command are stored\+: name of the command, pointer to \doxylink{classConsole}{Console} function that handles the command, and formattable description of the command.
