\doxysubsection{Detailed Description}
A class for printing data in a readable format. 

This class provides an opportunity to print in readable format any array of char (bytes) to any ostream, optionally showing the hexadecimal values of bytes.

Print data line be line replacing non-\/printable characters by \textquotesingle{}.\textquotesingle{}. If show\+\_\+hex is true, hex values are shown. Also, address provided in parameter "{}address"{} is printed in hex with offset in each line.
