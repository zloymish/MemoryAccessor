\doxylink{classConsole}{Console} source and Read\+Line integration. 

A source that contains the realization of \doxylink{classConsole}{Console} class and the integration of GNU Read\+Line library. Also, this file defines ctrl\+\_\+c\+\_\+pressed variable, which is used to detect if Ctrl-\/C was pressed when \doxylink{classConsole}{Console} class is used.
